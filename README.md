# About the mod ✨

This project is designed to support a battlemage playstyle by enhancing specific in-game items and expanding their functionality. It focuses on improving combat flexibility, allowing the player to make better use of elemental abilities and maintain a more consistent flow during battles.

The mod modifies several accessories, adding or improving effects related to stamina, spirit, and skill usage. It enables easier access to elemental attacks and helps create a more dynamic combat experience, where different elements can be used more freely and effectively.

# Main features ⚙️

- Adds enhanced buffs to key accessories
- Improves stamina and spirit management
- Expands access to elemental abilities
- Supports a more flexible battlemage build
- Includes multiple item-based effects for different playstyles
- Improves overall combat consistency and usability

# Gameplay impact 🎮

With this mod, combat becomes more responsive and less restrictive. Players can rely more on abilities rather than basic attacks, using a mix of elemental powers to handle different situations. The added buffs help maintain resources, reduce downtime, and improve the overall pacing of fights.

It is especially useful for players who prefer hybrid builds, combining magic and physical combat. The changes are designed to feel natural within the game while providing a noticeable improvement in performance and comfort during gameplay.

# Notes 📌

This mod is intended for players who want to experiment with a battlemage-style build and enhance their combat capabilities without changing the core mechanics of the game. It focuses on balance, usability, and a smoother gameplay experience.
